# Økt 2-2 — Variabler, aritmetiske operatorer og enkle if-setninger
**Dato:** onsdag 29. april 2026
**Tid:** ca. 1,5–3 timer

---

## Mål for økten
- Deklarere, tilordne og lese variabler med `let` og `const`
- Bruke aritmetiske operatorer
- Skrive enkle if-setninger med sammenligningsoperatorer

---

## Del 1 — Variabler (~30 %)

### Hva er en variabel?
En variabel er en **navngitt huskelapp** i programmet.

### Demo 1 — Deklarasjon og tilordning

Skriv i konsollen, steg for steg:

```js
// Deklarasjon: lage variabelen
let alder = 25;

// Lese: hva er verdien?
console.log(alder);       // 25

// Tilordning: endre verdien
alder = 26;
console.log(alder);       // 26
```

Poeng:
- `let` brukes når verdien kan endre seg
- `const` brukes når verdien ikke skal endre seg
- Variabelnavnet velger vi selv — bruk beskrivende navn

```js
const navn = "Kari";
navn = "Ole";  // FEIL — const kan ikke endres
```

### Demo 2 — Variabler på siden

```html
<p id="output"></p>

<script>
    let alder = 25;
    let navn = "Kari";

    document.getElementById("output").innerHTML =
        navn + " er " + alder + " år gammel.";
</script>
```

## Del 2 — Aritmetiske operatorer (~25 %)

```js
let a = 10;
let b = 3;

console.log(a + b);   // 13  — pluss
console.log(a - b);   // 7   — minus
console.log(a * b);   // 30  — gange
console.log(a / b);   // 3.33... — dele
```

### Demo 3 — Enkel kalkulator

```html
<input id="tall1" type="number" placeholder="Tall 1">
<input id="tall2" type="number" placeholder="Tall 2">
<button onclick="regnUt()">Regn ut</button>
<p id="svar"></p>

<script>
    function regnUt() {
        let tall1 = Number(document.getElementById("tall1").value);
        let tall2 = Number(document.getElementById("tall2").value);
        let resultat = tall1 + tall2;
        document.getElementById("svar").innerHTML = "Svar: " + resultat;
    }
</script>
```

Poeng:
- `Number(...)` konverterer tekst fra inputfeltet til et tall
- Uten `Number()` vil `"3" + "5"` bli `"35"` (tekstsammenslåing) ikke `8`

## Del 3 — If-setninger (~35 %)

### Hva er en if-setning?
Kode som bare kjøres *hvis* noe er sant.

### Demo 4 — Enkel if/else

```js
let alder = 20;

if (alder >= 18) {
    console.log("Du er voksen.");
} else {
    console.log("Du er ikke voksen ennå.");
}
```

### Sammenligningsoperatorer

| Operator | Betyr | Eksempel |
|---|---|---|
| `===` | er lik | `alder === 18` |
| `!==` | er ikke lik | `navn !== "Kari"` |
| `>` | større enn | `poeng > 50` |
| `<` | mindre enn | `poeng < 50` |
| `>=` | større enn eller lik | `alder >= 18` |
| `<=` | mindre enn eller lik | `alder <= 65` |

### Demo 5 — If på siden

```html
<input id="alderInput" type="number" placeholder="Skriv alder">
<button onclick="sjekkAlder()">Sjekk</button>
<p id="svar"></p>

<script>
    function sjekkAlder() {
        let alder = Number(document.getElementById("alderInput").value);

        if (alder >= 18) {
            document.getElementById("svar").innerHTML = "Du er voksen!";
        } else {
            document.getElementById("svar").innerHTML = "Du er ikke voksen ennå.";
        }
    }
</script>
```

### Demo 6 — if / else if / else

```js
function karakter(poeng) {
    if (poeng >= 90) {
        return "A";
    } else if (poeng >= 70) {
        return "B";
    } else if (poeng >= 50) {
        return "C";
    } else {
        return "Stryk";
    }
}
```

## Vanlige feil

| Feil | Årsak | Løsning |
|---|---|---|
| `"3" + "5"` gir `"35"` | Input er tekst, ikke tall | Pakk inn i `Number(...)` |
| `=` i stedet for `===` | Tilordning vs. sammenligning | `=` setter verdi, `===` sjekker verdi |
| `const` gir feil ved endring | `const` kan ikke tilordnes på nytt | Bruk `let` hvis verdien skal endre seg |

---

## Tips for de som henger etter
- Konsolltesting er nøkkelen — la dem skrive `let x = 5; x + 3` og se resultatet
- `Number()`-konverteringen er litt magisk — si bare "vi gjør dette fordi input alltid er tekst"
- Fokuser på `if/else` — `else if` er bonus
