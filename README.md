## Velkommen til Realkompetansekurset!

Dette kurset går over fem uker med to økter per uke. Du lærer å lage interaktive nettsider med HTML, CSS og JavaScript — og å jobbe i gruppe med en ekte case-oppgave.
Innlevering på fredager.

- Mandag, Onsdag · 9:00–10:00
- Fredag "Show and tell" 12:30 i Discord

Samme link hver gang: https://meet.google.com/gpn-kcxy-xis

**Ingen forkunnskaper kreves.**

---

## Pensum

- **HTML** — struktur: overskrifter, tekst, bilder, lenker, lister
- **CSS** — utseende: farger, fonter, Flexbox og Grid (overfladisk)
- **JavaScript**
  - Funksjoner (uten og med parametre, med returverdi)
  - Endre DOM: `getElementById` + `innerHTML`
  - Reagere på klikk: `onclick`
  - Produsere HTML fra JavaScript
  - Variabler: deklarasjon, tilordning, lesing
  - Operatorer: aritmetiske, tilordning, sammenligning, logiske
  - Enkle if-setninger
  - `null` og `undefined`
  - Enkel MVC-struktur

Vi bruker **ikke**: løkker, arrays eller objekter.
