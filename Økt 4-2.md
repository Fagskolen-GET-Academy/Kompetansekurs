# Økt 4-2 — Innebygde metoder: Tall
**Dato:** onsdag 13. mai 2026

---

## Mål for økten
- Kjenne til heltall og desimaltall i JavaScript
- Konvertere mellom tekst og tall med `Number()` og `parseInt()`
- Formatere tall med `toFixed()` og `toLocaleString()`
- Bruke de viktigste metodene fra `Math`-objektet

---

## Del 1 — Talltyper og operatorer (~15 %)

```js
let heltall = 42;
let desimal = 3.14;

// Modulus — resten etter heltallsdivisjon
console.log(10 % 3);   // 1
console.log(8 % 2);    // 0
```

Poeng:
- JavaScript skiller ikke mellom int og float — alt er `number`
- `%` brukes ofte til å sjekke partall: `if (tall % 2 === 0)`

---

## Del 2 — Konvertering: tekst → tall (~25 %)

### `Number()` vs `parseInt()`

```js
Number("42")         // 42
Number("3.14")       // 3.14
Number("42px")       // NaN

parseInt("42")       // 42
parseInt("3.14")     // 3    — kutter desimalene
parseInt("42px")     // 42   — leser så langt det går
parseInt("0xFF", 16) // 255  — støtter andre tallsystemer
```

Poeng:
- Bruk `Number()` når du forventer et rent tall
- Bruk `parseInt()` når du vil ha heltall, eller input kan ha tekst etter tallet

---

## Del 3 — Konvertering: tall → tekst + formatering (~25 %)

```js
let pris = 1234567.891;

pris.toFixed(2)
// "1234567.89"

pris.toLocaleString("nb-NO", { style: "currency", currency: "NOK" })
// "1 234 567,89 kr"

pris.toLocaleString("nb-NO")
// "1 234 567,891"
```

Poeng:
- `toFixed()` returnerer en **streng**, ikke et tall
- `"nb-NO"` er koden for norsk bokmål

Referanse: https://www.w3schools.com/jsref/jsref_tolocalestring_number.asp

## Del 4 — Math-objektet (~25 %)

```js
Math.abs(-5)        // 5
Math.round(4.6)     // 5
Math.ceil(4.1)      // 5    — runder alltid opp
Math.floor(4.9)     // 4    — runder alltid ned
Math.min(3, 1, 7)   // 1
Math.max(3, 1, 7)   // 7
Math.pow(2, 8)      // 256
Math.random()       // 0.0–0.999...
```

### Tilfeldig heltall mellom to verdier
```js
function tilfeldigHeltall(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}

tilfeldigHeltall(1, 6)   // terningkast
```

Referanse: https://www.w3schools.com/jsref/jsref_obj_math.asp

## Vanlige feil

| Feil | Årsak | Løsning |
|---|---|---|
| `toFixed()` gir feil ved videre regning | Returnerer streng | Pakk inn i `Number()` |
| `parseInt("3.14")` gir 3 | Kutter desimaler | Bruk `Number()` for desimaltall |
| `Math.random()` gir aldri heltall | Returnerer desimaltall | Kombiner med `Math.floor()` |
| `NaN` dukker opp | Ugyldig konvertering | Sjekk input med `isNaN()` |

---

## Tips for de som henger etter
- Konsollen er perfekt her — la dem bare prøve metodene selv
- `round/ceil/floor`: vis alle tre på samme tall side om side
- `toLocaleString` med norsk format motiverer — ser profesjonelt ut med én linje
