## Del 1 — Talltyper og operatorer (~10 %)

---

## Del 2 — Konvertering: tekst → tall (~15 %)

```js
Number("42")         // 42
Number("3.14")       // 3.14
Number("42px")       // NaN

parseInt("42")       // 42
parseInt("3.14")     // 3    — kutter desimalene
parseInt("42px")     // 42   — leser så langt det går
parseInt("0xFF", 16) // 255  — støtter andre tallsystemer
```

Tommelfingerregel: bruk `Number()` for rene tall, `parseInt()` når input kan inneholde tekst etter tallet.

---

## Del 3 — Formatering: tall → tekst (~15 %)

```js
let pris = 1234567.891;

pris.toFixed(2)
// "1234567.89"

pris.toLocaleString("nb-NO", { style: "currency", currency: "NOK" })
// "1 234 567,89 kr"
```

Poeng: `toFixed()` returnerer en **streng**, ikke et tall. Pakk inn i `Number()` hvis du skal regne videre.

Referanse: https://www.w3schools.com/jsref/jsref_tolocalestring_number.asp

## Del 4 — Math-objektet (~15 %)

```js
Math.abs(-5)        // 5
Math.round(4.6)     // 5
Math.ceil(4.1)      // 5    — runder alltid opp
Math.floor(4.9)     // 4    — runder alltid ned
Math.min(3, 1, 7)   // 1
Math.max(3, 1, 7)   // 7
Math.pow(2, 8)      // 256
Math.random()       // 0.0–0.999...
```

```js
// Tilfeldig heltall mellom to verdier
function tilfeldigHeltall(min, max) {
    return Math.floor(Math.random() * (max - min + 1)) + min;
}
```

## Del 5 — Strenger er uforanderlige (~5 %)

Viktig poeng før vi ser på metoder:

```js
let navn = "  Kari  ";
navn.trim();           // returnerer "Kari"
console.log(navn);     // "  Kari  " — uendret!

let ryddetNavn = navn.trim();  // lagre resultatet
```

## Del 6 — Strengmetoder (~30 %)

### Informasjon om strengen
```js
let tekst = "Hei, verden!";

tekst.length              // 12
tekst.includes("verden")  // true
tekst.startsWith("Hei")   // true
tekst.endsWith("!")       // true
tekst.indexOf("verden")   // 5
```

### Endre og hente deler
```js
tekst.toUpperCase()              // "HEI, VERDEN!"
tekst.toLowerCase()              // "hei, verden!"
tekst.trim()                     // fjerner mellomrom i endene
tekst.slice(5, 11)               // "verden"
tekst.replace("verden", "JS")    // "Hei, JS!"
tekst.replaceAll("e", "a")       // "Hai, vardan!"
tekst.repeat(2)                  // "Hei, verden!Hei, verden!"
```

### Tegn og deling
```js
tekst.charAt(0)          // "H"
tekst.charCodeAt(0)      // 72

let csv = "Kari,Ole,Fatima";
csv.split(",")           // ["Kari", "Ole", "Fatima"]
```

## Vanlige feil

| Feil | Årsak | Løsning |
|---|---|---|
| `toFixed()` gir feil ved videre regning | Returnerer streng | Pakk inn i `Number()` |
| `parseInt("3.14")` gir 3 | Kutter desimaler | Bruk `Number()` for desimaltall |
| Strengmetode "endrer ikke" strengen | Strenger er uforanderlige | Lagre resultatet i en variabel |
| `indexOf()` gir -1 | Teksten ikke funnet | -1 betyr "finnes ikke" |

## Tips

- Ikke gå gjennom alle strengmetodene — fokuser på `trim()`, `toUpperCase()`,`toLowercase()` `includes()`, `replace()`, `indexOf()` og `slice()`
