## Del 2 — HTML (~45 %)

### Hva er en nettside?
Åpne DevTools (F12) på en hvilken som helst side og vis at alt er HTML under panseret.

> Spørsmål: "Hva tror dere nettsider er laget av?"

### Demo 1 — Den enkleste mulige nettsiden
```html
<!DOCTYPE html>
<html lang="no">
<head>
    <meta charset="UTF-8">
    <title>Min side</title>
</head>
<body>
    Hei verden!
</body>
</html>
```

Gå gjennom alle linjene én etter én. Ikke rush.

### Demo 2 — Overskrifter, tekst, bilde, lenke, liste
Bygg opp siden gradvis:
```html
<h1>Navn</h1>
<p>Et avsnitt om meg.</p>
<ul>
    <li>Hobby 1</li>
    <li>Hobby 2</li>
</ul>
<img src="https://via.placeholder.com/200x200" alt="Bilde">
<a href="https://getacademy.no">GET Academy</a>
```

### Vanlige feil 
| Feil | Hva skjer | Løsning |
|---|---|---|
| Glemme `</h1>` | Resten av siden blir overskrift | Lukk alle tagger |
| Bilde vises ikke | Feil filsti | Sjekk at filnavn stemmer |

## Del 3 — Enkel CSS (~30 %)

### Demo 3 — Koble til CSS og style siden
```html
<link rel="stylesheet" href="style.css">
```

```css
body {
    background-color: #f0f4f8;
    font-family: Arial, sans-serif;
    color: #333;
    padding: 32px;
}

h1 {
    color: #005f99;
}

.ingress {
    font-size: 20px;
    font-style: italic;
}
```

Poeng:
- `<link>` kobler CSS-filen til HTML — filnavnet må stemme nøyaktig
- Klasser starter med `.` i CSS og `class=""` i HTML
- Bruk DevTools til å eksperimentere med farger live
